UZASADNIENIE

Klipy proxy używane podczas edycji proxy mają zazwyczaj mniejszą szybkość transmisji danych i mniej intensywne dekodowanie procesora. Istnieją dwa główne powody używania edycji proxy:
Oryginalne nośniki, z których program jest edytowany, powodują zbyt wysokie zapotrzebowanie na przepustowość dysku lub moc przetwarzania procesora w celu dekodowania, aby umożliwić elastyczną edycję
Oryginał przechowywany jest na serwerze sieciowym, wolnym dysku zewnętrznym lub innym nośniku z ograniczonym dostępem i nie umożliwia szybkiej edycji w przypadku bezpośredniego dostępu.

OGÓLNY OBIEG PRACY DLA PROXY

Wszystkie przepływy pracy przy edycji proxy mają te same fazy:

  • Renderuj media proxy z oryginalnych nośników

  • Zamień oryginalne nośniki na media proxy

  • Edytuj za pomocą mediów proxy

  • Zamień nośnik proxy na oryginalny nośnik

  • Renderuj końcowy program przy użyciu oryginalnych nośników

    Tworzenie Proxy Media
    Wybierz Project-> Proxy Manager w menu i ustaw ustawienia plików proxy w obszarze Kodowanie Proxy
    Wybierz pliki wideo mediów w panelu Media
    Naciśnij przycisk z ikoną pliku proxy następnie przycisk Usuń w panelu Media Jeśli projekt jest już w trybie proxy "Korzystanie z proxy proxy" , klipy na osi czasu z oryginalnymi nośnikami, na których były wyświetlane media proxy, zostaną natychmiast zastąpione przez media proxy
    Konwertowanie na nośnik proxy
    Wybierz Project-> Proxy Manager, aby otworzyć Proxy Manager
    Naciśnij przycisk Użyj media proxy
    Edytowanie za pomocą mediów proxy
    Klipy korzystające z mediów proxy mają niebieski pasek wskazujący ten status Ikona wskaźnika edycji proxy jest wyświetlana w lewym dolnym rogu
    Jeśli nowe nośniki proxy zostaną renderowane, klipy osi czasu z użyciem oryginalnego nośnika zostaną natychmiast zastąpione nośnikami proxy
    Projekt można zapisać normalnie i przekonwertować po załadowaniu, aby użyć oryginalnych nośników
    Konwertowanie na oryginalne Media
    Wybierz Project-> Proxy Manager, aby otworzyć Proxy Manager
    Naciśnij przycisk Użyj oryginalnego nośnika
